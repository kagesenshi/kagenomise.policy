kagenomise.policy Installation
------------------------------

To install kagenomise.policy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``kagenomise.policy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        kagenomise.policy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
